# Isuzu's Shaders

## 概要

アバターに適用することを目的としたトゥーンシェーダー。

## 機能

### シェーダー

* 基本的なテクスチャ表示
* 視差マッピング
* リム
* エミッション
* スペキュラマップ
* アウトライン
* オクルージョンマップ
* ディストーション
* レンダーテクスチャ
* ステンシル
* テッセレーション
* 両面描画
